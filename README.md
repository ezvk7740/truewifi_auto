# TrueWifi auto-connector

## Setup

```
chmod +x setup.sh
./setup.sh
```
add your username & password to credentials.txt

## Running

change to root user
`sudo -s` or `su`

then run 
Debian
```
./main_debian.sh
```
Ubuntu
```
./main_ubuntu.sh
```

if you dont have root access, just remove every `sudo` out of the script
